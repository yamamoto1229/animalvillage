# <アプリ名>
どうぶつの里

## サイト概要
保護動物を対象とした
動物の里親を見つけるサイト
動物保護センターと一般ユーザーとの交流サイト

### サイトテーマ
保護動物の里親を見つけるサイト
動物保護センター情報観覧サイト

### テーマを選んだ理由
犬や猫の年間殺処分の記事を読んだ時に
この数をどうにか減らせないかと思い
動物保護センターと一般ユーザーとの交流サイトがあると
その数は減らせるのではないかと思った。

### ターゲットユーザ
動物を愛する全てのユーザー
動物保護センター

### 主な利用シーン
保護動物を引き取りたいがどうすればいいかわからない人への
情報提供と保護センターとの仲介役
動物保護センターの新着情報掲載
